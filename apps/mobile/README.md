# SnapEat Mobile App

React Native mobile application built with Expo and TypeScript.

## Features

- User Authentication
- Food photo scanning
- Meal logging
- Progress analytics
- Profile management

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on iOS:
```bash
npm run ios
```

4. Run on Android:
```bash
npm run android
```

## Tech Stack

- React Native
- Expo
- Expo Router (File-based routing)
- TypeScript
- React Query
- Zustand (State management)
- Expo Camera
- Expo Image Picker
